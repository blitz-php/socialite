## Installation

Vous pouvez installer le package via composer :

```bash
composer require blitz-php/socialite
```

## Utilisation

```php
$skeleton = new Blitz-php\Socialite();
echo $skeleton->echoPhrase('Hello, Blitz-php!');
```
